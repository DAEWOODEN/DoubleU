# 🚀 ComChatX 最简单部署指南（旧版，已废弃）

## ⚡ 3步完成部署（5分钟）

### 步骤1: 安装ngrok（只需一次）

在终端运行：

```bash
brew install ngrok/ngrok/ngrok
```

如果没有Homebrew，访问 https://ngrok.com/download 下载。

---

### 步骤2: 注册ngrok账号（可选但推荐）

**为什么？** 注册后URL更稳定，不会每次都变化。

1. 访问 https://dashboard.ngrok.com/signup
2. 注册账号（可以用GitHub登录）
3. 在Dashboard获取你的authtoken
4. 在终端运行：

```bash
ngrok config add-authtoken YOUR_TOKEN
```

（替换 `YOUR_TOKEN` 为你的实际token）

---

### 步骤3: 一键启动

**最简单方式：**

在终端进入项目目录，运行一键脚本：

```bash
cd ~/Desktop/ComChatX
./快速部署.sh
```

**或者手动启动（如果需要更多控制）：**

#### Terminal 1 - 启动后端：
```bash
cd ~/Desktop/ComChatX/backend
source venv/bin/activate
python3 main.py
```

#### Terminal 2 - 启动前端：
```bash
cd ~/Desktop/ComChatX/frontend
npm run dev
```

#### Terminal 3 - 启动ngrok：
```bash
ngrok http 3000
```

---

## 📝 常见问题

### 1. 命令找不到？

**问题：** `python: command not found`

**解决：** 使用 `python3` 代替 `python`

```bash
python3 main.py
```

### 2. 找不到venv？

**问题：** `source: no such file or directory: venv/bin/activate`

**解决：** 先创建虚拟环境：

```bash
cd ~/Desktop/ComChatX/backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 3. 找不到node_modules？

**问题：** 前端启动失败

**解决：** 先安装依赖：

```bash
cd ~/Desktop/ComChatX/frontend
npm install
npm run dev
```

### 4. 端口被占用？

**问题：** `Address already in use`

**解决：** 停止占用端口的进程：

```bash
# 停止8000端口
lsof -ti:8000 | xargs kill -9

# 停止3000端口
lsof -ti:3000 | xargs kill -9
```

### 5. ngrok连接失败？

**检查：**
1. 本地服务是否正常运行？
2. 端口是否正确（前端3000，后端8000）？
3. 防火墙是否阻止？
