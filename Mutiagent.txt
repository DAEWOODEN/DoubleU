
# ComchatX

---

## 一、产品概述

### 1.1 产品定位
一个基于Multi-Agent架构的智能留学文书创作平台，通过对话式交互收集用户碎片化想法，经过AI深度分析、质疑、引导和挖掘，形成深入的自我认知，清晰完整的自我叙事并且最终生成高质量的个人陈述（Personal Statement）。

### 1.2 核心价值
- **想法收集**：随时记录零散灵感，极简优雅的"想法球"交互
- **思维引导**：AI 主动提问深挖潜力，苏格拉底式追问
- **叙事架构**：构建完整自我认知框架，可视化成长轨迹
- **文书创作**：多学校差异化生成，专业级个人陈述
- **长期陪伴**：持续迭代优化，记录成长时刻

### 1.3 核心模式

- 零散想法收集 → 深入自我认知

- 碎片化信息 → 完整自我叙事

- 智能引导 → 高质量个人陈述

## 三、Multi-Agent 系统设计

### 3.1 Agent 角色与职责详解

#### Agent 0: 全局管理者 (Global Supervisor Agent)

**核心职责**:
- 用户对话接口
- 自主统筹调度所有 SubAgent 的工作流程
- 维护对话上下文和用户画像
- 动态生成专业领域 SubAgent
- 锚定三大核心目标: 自我认知 → 自我叙事 → 高质量文书

---

#### Agent 1: 收集者 (Collector SubAgent)

**核心职责**:
- 收集用户初始信息
- 接收用户零散想法输入
- 接收用户对话内容输入

---

#### Agent 2: 分析者 (Analyzer SubAgent)

**核心职责**:
- 每收集 5 条新想法触发一次分析
- 分析用户对话内容输入
- 发现潜在优势和亮点
- 识别想法之间的关联性
- 生成完整性报告给 Global Agent

---

#### Agent 3: 引导者 (Guide SubAgent)

**核心职责**:
- 根据 Analyzer 报告生成针对性问题
- 深度挖掘浅层想法）
- 补全缺失类别的信息
- 采用开放式提问，避免 yes/no
- 人性化语气，避免机械感


---

#### Agent 4: 质疑者 (Query SubAgent)

**核心职责**:
- 苏格拉底式追问，挑战用户思维
- 发现逻辑漏洞和表面化描述
- 推动更深层次的自我认知
- 用户可选择停止质疑


---

#### Agent 5: 叙事者 (Narrator SubAgent)

**核心职责**:
- 将零散想法组织成结构化叙事框架
- 提炼 3-5 个核心主题
- 构建时间线和因果关系
- 生成"起承转合"故事大纲
- 输出可视化叙事图谱

---

#### Agent 6: 写作者 (Writer SubAgent)

**核心职责**:
- 基于叙事框架生成文书初稿
- 支持多版本风格（学术/故事/简洁）
- 流式生成，实时展示
- 针对不同学校个性化调整
- 段落级优化和润色

**版本控制**:
- 自动保存每次生成的版本
- 支持版本对比(diff 高亮)
- 用户可恢复任意历史版本

---

#### Agent 7: 审核者 (Audit SubAgent)

**核心职责**:
- 质量评估(可读性、词汇多样性、情感强度)
- 原创性检测(避免套话)
- 格式规范检查
- 提供具体改进建议
- 最终把关

---

#### Agent N: 动态专业领域 Agent (Major SubAgents)

**生成逻辑**:
- 当 Collector 发现用户提到某专业领域时触发
- Global Agent 动态创建对应专业的 SubAgent
- 专业 Agent 提供该领域的专业知识支持

**专业 Agent 职责**:
1. **术语准确性**: 检查专业术语使用是否正确
2. **深度评估**: 判断专业描述是否有深度
3. **建议补充**: 提示可以补充的专业亮点
4. **审核把关**: 最终审核专业相关段落
